# BORG Minecraft Launcher

---

## Описание

BORG Minecraft Launcher - современный лаунчер для Minecraft с поддержкой модпаков Modrinth (.mrpack). Лаунчер написан на Rust с использованием Tauri, что обеспечивает высокую производительность и надежность.

---

## Основные возможности

### Управление модами
- Автоматическая синхронизация модов из Modrinth MRpack файлов
- Проверка целостности файлов по SHA1 хешам
- Пропуск уже скачанных модов для экономии трафика
- Извлечение файлов из папки overrides

### Настройки игры
- Выбор версии Minecraft
- Настройка объема оперативной памяти (RAM)
- Указание пути к игре и Java
- Сохранение настроек между запусками

### Интерфейс
- Современный темный дизайн
- Отображение новостей из GitHub
- Log с прогресс-баром синхронизации
- Копирование и очистка лога
- Системный трей для быстрого доступа

## Установка

### Требования
- Windows 10 или новее
- Java 17 или новее (для запуска Minecraft)
- Интернет-соединение для синхронизации модов

### Запуск
1. Скачайте последнюю версию из раздела Releases
2. Распакуйте архив в любую папку
3. Запустите `BORG-Launcher.exe`

---

## Использование

### Первая настройка
1. Укажите ваш никнейм
2. Выберите путь к папке игры
3. Укажите путь к Java (если не найден автоматически)
4. Настройте объем RAM для игры
5. Нажмите кнопку "Play" для запуска

### Синхронизация модов
1. Нажмите кнопку "Sync" для синхронизации модов
2. Моды будут скачаны из MRpack файла
3. Прогресс отображается в Battle Log
4. Уже скачанные моды пропускаются

### Настройки
- Нажмите кнопку настроек (иконка шестеренки)
- Измените необходимые параметры
- Нажмите "Save" для сохранения

## Файлы и папки

### Структура проекта
```
BORGLauncher/
├── tauri-launcher/          # Основной код лаунчера
│   ├── src/                 # Frontend (HTML/CSS/TypeScript)
│   └── src-tauri/           # Backend (Rust)
├── config.json              # Конфигурация лаунчера
├── news.md                  # Локальный файл новостей
└── README.md                # Этот файл
```

### Конфигурация
- `config.json` - настройки лаунчера (никнейм, RAM, пути)
- `news.md` - локальная копия новостей (загружается из GitHub)

## Troubleshooting

### Моды не скачиваются
- Проверьте интернет-соединение
- Убедитесь, что MRpack файл доступен на GitHub
- Проверьте Battle Log на наличие ошибок

### Игра не запускается
- Убедитесь, что Java установлена и работает
- Проверьте путь к Java в настройках
- Убедитесь, что версия Minecraft установлена

### Ошибка 404 при загрузке новостей
- Файл новостей может отсутствовать на GitHub
- Лаунчер автоматически использует локальную копию
